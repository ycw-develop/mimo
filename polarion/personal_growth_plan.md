## 四、C语言学习计划（12周）

### 学习资源

- **入门书**：《C Primer Plus》第6版
- **视频**：B站搜 "C语言入门"（翁恺/浙大 或 鹏哥C语言）
- **刷题**：牛客网C语言入门题 / LeetCode简单题
- **工具**：VS Code + GCC 或 CLion

### 阶段一：基础语法（第1-3周）

**第1周：环境搭建 + 基本语法**
- 安装开发环境（VS Code + GCC）
- 变量、数据类型（int, float, char, double）
- printf / scanf 输入输出
- 运算符（算术、关系、逻辑）
- 练习：写5个小程序（计算器、温度转换等）

**第2周：控制结构**
- if / else / switch
- for / while / do-while 循环
- break / continue
- 练习：九九乘法表、求素数、水仙花数

**第3周：函数 + 数组**
- 函数定义、调用、参数传递
- 一维数组、二维数组
- 数组遍历、排序（冒泡排序）
- 练习：数组求最大值、排序

### 阶段二：指针与内存（第4-6周）⚠️ 核心难点

**第4周：指针入门**
- 什么是指针、指针变量
- 取地址 &、解引用 *
- 指针与数组的关系
- 画内存图理解（**这步不能跳**）

**第5周：指针进阶**
- 指针运算
- 指针与函数（指针作为参数）
- 字符串与字符指针
- 多级指针（了解即可）

**第6周：内存管理**
- malloc / free 动态内存分配
- 内存泄漏的概念
- 结构体（struct）
- 结构体指针
- 练习：实现一个简单的通讯录

### 阶段三：数据结构基础（第7-9周）

**第7周：链表**
- 单链表：创建、插入、删除、遍历
- 用结构体 + 指针实现
- 练习：手写链表增删改查

**第8周：栈和队列**
- 栈：数组实现、链表实现
- 队列：数组实现、链表实现
- 练习：用栈实现括号匹配

**第9周：综合练习**
- 综合前面所学，完成一个小项目
- 推荐：学生管理系统 / 简易银行系统
- 文件读写（fopen, fread, fclose）

### 阶段四：结合实际（第10-12周）

**第10周：看甲方代码**
- 争取拿到甲方项目的代码（哪怕是部分）
- 读代码、理解结构、做笔记
- 不懂的标注出来请教同事或领导

**第11周：尝试参与**
- 主动问领导有没有简单的C任务
- 从改bug或写简单功能开始
- 继续刷题保持手感

**第12周：复盘总结**
- 整理学习笔记
- 总结哪些掌握了、哪些还薄弱
- 制定下一阶段计划
